Tutorial Screen

What can creators do?

In this module starter, creators can write a message which is shown to their audience, and record a video of their camera.

You'll see these output as assets in the SDK, with the name fullScreenRecording for the video, and replay for the data with the message.

What can audiences do?

Audiences can then see the message written by the creator, and record a video of their own to send back to the creator.

You'll need to set the creator's output asset replay as an audience input asset with the same name.
